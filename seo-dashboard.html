🔴
Très peu de contenu / articles (les 2 sites)
prepamedecine.fr a ~7 pages, afem-edu.fr a ~8 pages. Trop peu pour que Google considère ces sites comme des autorités thématiques. Objectif : 30+ pages par site.
CRITIQUE
🔴
Pas de sitemap.xml (prepamedecine.fr)
Aucun sitemap détecté. Google ne peut pas découvrir efficacement vos pages. Créez un sitemap.xml et soumettez-le à Google Search Console.
CRITIQUE
🔴
Pas de robots.txt (prepamedecine.fr)
Le fichier robots.txt n'existe pas. Google utilise ce fichier pour comprendre la structure de votre site.
CRITIQUE
🔴
Pas de Schema markup (prepamedecine.fr)
Aucun JSON-LD/Schema détecté. Ajoutez au minimum Organization + FAQPage pour améliorer l'affichage dans les résultats Google.
CRITIQUE
🟡
Meta descriptions manquantes ou génériques
Certaines pages n'ont pas de meta description optimisée. Chaque page doit avoir une description unique de 150-160 caractères avec le mot-clé cible.
WARNING
🟡
Alt text manquants sur les images (les 2 sites)
Les images n'ont pas d'attribut alt descriptif. Impact sur l'accessibilité et le SEO images.
WARNING
🟡
URLs en /index.php/ (prepamedecine.fr)
Les URLs contiennent /index.php/ (ex: /index.php/diploma-sante/). Configurez les permaliens WordPress pour avoir des URLs propres (/diploma-sante/).
WARNING
🟡

La section blog existe dans le menu mais pas de contenu régulier détecté. Publiez 2-3 articles par semaine minimum.
