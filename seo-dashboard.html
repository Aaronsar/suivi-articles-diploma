Problèmes SEO
🔴
Pas de sitemap.xml
Le fichier sitemap.xml retourne une erreur 404. WordPress peut le générer automatiquement via Yoast SEO ou Rank Math. Indispensable pour que Google découvre toutes les pages.
CRITIQUE
🔴
Pas de robots.txt
Le fichier robots.txt n'est pas détecté. WordPress le génère normalement automatiquement. Vérifier la configuration ou le créer manuellement.
CRITIQUE
🔴
Aucun Schema JSON-LD détecté
Pas de données structurées (Organization, LocalBusiness, Course, FAQPage). Ajouter via un plugin WordPress comme Rank Math ou Schema Pro. Très important pour les rich snippets Google.
CRITIQUE
🟡
Meta descriptions possiblement manquantes
La meta description de la homepage n'a pas été détectée. Vérifier toutes les pages avec un plugin SEO (Yoast/Rank Math). Chaque page doit avoir une meta description unique de 150-160 caractères.
WARNING
🟡
URLs dupliquées possibles
/prepa-terminale et /preparations/terminale-sante/ semblent être deux pages similaires. /premiere-sante/ et /preparations/premiere-elite/ aussi. Risque de cannibalisation SEO — vérifier et rediriger si nécessaire.
WARNING
🔵
Bonne couverture des mots-clés par université
Les pages par université (Sorbonne, Paris-Cité, Saclay, UPEC, SPN) couvrent bien les recherches géo-ciblées comme "prépa PASS Sorbonne".
BON
🔵
Contenu riche et bien structuré
Pages de présentation détaillées avec stats de réussite (78% d'admission), témoignages, FAQ, et détails des formations. Bonne base de contenu.
